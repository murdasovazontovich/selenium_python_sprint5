Selenium UI tests for Stellar Burgers

Проект содержит автотесты пользовательского интерфейса сайта **Stellar Burgers**, написанные с использованием **Selenium WebDriver** и **pytest**.

Тесты покрывают:
- регистрацию пользователя;
- авторизацию пользователя;
- переходы между разделами конструктора бургера (Булки, Соусы, Начинки);
- выход из личного кабинета.

---

 Технологии
- Python 3.9+
- Selenium WebDriver
- pytest
- Google Chrome / ChromeDriver
